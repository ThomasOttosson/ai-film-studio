# Support Lifecycle

## Purpose

This document defines the support lifecycle for software components, services, and releases within AI Film Studio.

## Lifecycle States

### Planned
Work has been approved but is not yet available.

### Active
Feature or service is fully supported with fixes, maintenance, and security updates.

### Maintenance
Only security fixes, critical bug fixes, and compliance updates are provided.

### Deprecated
The component remains available but is scheduled for removal. New development should avoid relying on it.

### End of Support
No further fixes or updates are provided. Migration should be completed.

### Retired
The component has been removed from service.

## Deprecation Policy

Deprecations should include:

- Reason
- Affected components
- Recommended replacement
- Migration guidance
- Expected retirement date

## Responsibilities

Engineering teams are responsible for:

- Communicating lifecycle changes
- Maintaining migration documentation
- Monitoring remaining usage
- Removing unsupported dependencies
