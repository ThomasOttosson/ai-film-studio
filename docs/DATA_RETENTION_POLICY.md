# Data Retention Policy

This document defines default data retention practices for AI Film Studio.

## Objectives

- Retain data only as long as necessary.
- Meet legal and contractual obligations.
- Minimize storage of unnecessary personal information.
- Support secure deletion and recovery processes.

## Recommended Retention

| Data Type | Recommended Retention |
|-----------|-----------------------|
| Application logs | 90 days |
| Audit logs | 1 year |
| Security events | 1 year or longer if required |
| Crash reports | 180 days |
| Backups | According to Backup & Restore policy |
| User-generated projects | Until deleted by the user or account closure policy |

## Secure Deletion

- Delete data using approved deletion procedures.
- Ensure expired backups are removed according to schedule.
- Revoke access to deleted resources.
- Verify deletion jobs through monitoring.

## Privacy

Personal data should only be retained when necessary for providing the service, complying with legal obligations, resolving disputes, or enforcing agreements.

## Review

Review this policy annually and after significant legal, regulatory, or platform changes.
