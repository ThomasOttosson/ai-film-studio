# Browser Support Policy

## Supported Browsers

AI Film Studio supports the latest stable releases of:

- Google Chrome
- Microsoft Edge
- Mozilla Firefox
- Apple Safari

## Support Window

- Chrome: latest 2 major versions
- Edge: latest 2 major versions
- Firefox: latest 2 major versions
- Safari: latest 2 major versions on supported macOS/iOS releases

## JavaScript

The application requires modern JavaScript (ES2022+) and does not support legacy browsers such as Internet Explorer.

## Progressive Enhancement

Core functionality should remain accessible where practical, while advanced capabilities such as realtime collaboration may require modern browser APIs.

## Testing

Before each production release, verify:
- Authentication
- Editor
- Media upload
- Rendering workflow
- Realtime collaboration
- Responsive layout
- Accessibility checks

Browser support should be reviewed at least twice per year.
